# Safe File Cleaner（GUI版）

Pythonで作成した **安全に不要ファイルを削除できるGUIツール** です。
指定したフォルダ内から、指定した拡張子のファイルのみを削除できます。

誤削除を防ぐため、**削除対象を限定できる設計**にしています。

---

## ✅ 主な機能

* 指定フォルダ内のファイルを安全に削除
* 削除する拡張子を複数指定可能（例：`.txt,.log,.tmp`）
* 削除結果を実行ログにリアルタイム表示
* 日本語対応のGUI操作
* Python標準ライブラリのみ使用（tkinter / pathlib）

---

## 🖥 動作環境

* macOS
* Python 3.11 〜 3.14
* Homebrew版 Python 推奨
* tkinter 使用

---

## ▶ 使い方

### ① 仮想環境の作成と起動

```bash
python3 -m venv .venv
source .venv/bin/activate
```

---

### ② アプリの起動

```bash
python safe_file_cleaner_gui.py
```

---

### ③ 操作手順

1. 「削除対象フォルダ」に対象フォルダのパスを入力
2. 「削除する拡張子」に削除したい拡張子を入力
   例：

   ```
   .txt,.log,.tmp
   ```
3. 「削除実行」ボタンを押す
4. 削除結果が実行ログに表示されます

---

## ⚠ 注意事項

* **指定した拡張子のファイルは完全削除されます**
* 実行前に **必ずバックアップを取ってください**
* フォルダパスの入力ミスに注意してください
