# Safe File Cleaner（安全なファイル削除ツール）

指定したフォルダ内の不要ファイルを、拡張子指定で安全に削除できるPythonツールです。  
誤削除防止の確認機能つき。

---

## 🔹 主な機能

- 指定フォルダ内のファイルを拡張子で抽出
- 複数拡張子に対応（例：.txt, .log, .tmp）
- 削除前に対象ファイルを一覧表示
- 「yes」入力時のみ削除実行（誤操作防止）
- pathlib使用でWindows / Mac 両対応

---

## 🔹 使用環境

- Python 3.8 以上
- 標準ライブラリのみ使用（追加インストール不要）

---

## 🔹 使い方

### ① スクリプトを実行

```bash
python safe_file_cleaner.py
